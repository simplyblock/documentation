Security is a core pillar of the simplyblock platform, designed to protect data across every layer of the storage
stack. From encryption at rest to multi-tenant isolation and secure communications, simplyblock provides robust,
enterprise-grade features that help meet stringent compliance and data protection requirements. Security is
enforced by design, ensuring your workloads and sensitive data remain protected against internal and external
threats.

This section covers the following security topics:

- **[NVMe-oF Connection Security](nvmeof-security.md):** Manage allowed host lists, configure DHCHAP authentication
  (unidirectional and bidirectional), and enable TLS with pre-shared keys for encrypted NVMe-oF connections.
- **[Volume Encryption](encryption-kubernetes-secrets.md):** AES-XTS encryption at rest for logical volumes.
- **[Multi-Tenancy](multi-tenancy.md):** Tenant isolation and segregation of encryption keys, metadata, and data.
